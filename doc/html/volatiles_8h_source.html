32 /*
33  * Some templated functions to handle setting, getting and comparing volatiles
34  * Not needed to 8 bit quantities but essential for 16 and 32 bit quantities
35  * The technique is simply to wrap the operation with calls to
36  * enable and disable interrupts.
37  *
38  * Note that there is still no absolute guarantee this will prevent errors
39  * cause by accessing volatiles.
40  *
41  * Wherever possible, provide other ways to handle quantities that can be
42  * modified inside interrupt service routines.
43  *
44  * Also note that this is a quick hack rather than a comprehensive and carefully
45  * considered solution to a common problem.
46  */
47
